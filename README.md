# Game of life

This is a simple implementation of Conway's Game of Life using C++ and the Windows API. 
The Game of Life is a cellular automaton that simulates the evolution of cells based on a set of rules.

## Controls

| Button   | Label           |
|----------|-----------------|
| Space    | Start/Pause     |
| R        | Generate Random |
| Arrow Up | Speed Up        |
| Arrow Up | Speed Down      |
| C        | Clear Field     |
| ESC      | Close           |
